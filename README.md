## ✨ Features

| Feature | Description |
|--------|-------------|
| 📦 Ground Items | Shows item info when approaching dropped items in the world |
| 🏪 Shop Terminals | Shows item info when hovering vending machines and bazaar terminals |
| 🎲 Artifact of Command | Shows item info when hovering items in the command picker |

---

## 📋 Item Info Panel

Each panel displays:
- **Item name** and rarity
- Item description
- Detailed stats with per-stack values

---

## 📦 Installation

### r2modman / Thunderstore Mod Manager *(recommended)*
1. Open r2modman and select **Risk of Rain 2**
2. Go to **Online** and search for `Too Many Items`
3. Click **Install**

### Manual
1. Install [BepInEx](https://thunderstore.io/package/bbepis/BepInExPack/)
2. Place `Too Many Items.dll` in `BepInEx/plugins/`
